# data-management-workshop
(trial for workshop)
This is a developmental, testing and preparation repository for the Data Managements Tools Catalogue
(the final and official one is under Open-Science-Department-KNAV account and in its own new repository). This one serves for experiments with code, in order to keep the history of the production repository more clean and organized.
More about the catalogue: https://openscience.lib.cas.cz/en/open-fair-data/catalogue-of-data-management-tools/.
Production website for the catalogue: https://open-science-department-knav.github.io/Data-Management-Tools-Catalogue/.
